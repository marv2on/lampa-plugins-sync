rating_light.js — Плагін для Lampa, який підтягує рейтинги Rotten Tomatoes та Metacritic через зовнішній API OMDb (потрібен свій API-ключ). Відображає їх як додаткові блоки поряд з IMDB/KP. Колір цифр — динамічний градієнт від червоного (0) → жовтого (5) → зеленого (10).
​

cubcolor.js — Плагін для Lampa, який рахує CUB-рейтинг на основі реакцій користувачів (🔥 fire, 👍 nice, 🤔 think, 😴 bore, 💩 shit) з вагами (10 / 7.5 / 5 / 2.5 / 0). Використовує формулу Байєса з середнім по базі (окремо для фільмів і серіалів). Мінімум 20 реакцій для відображення. Колір блоку — той самий градієнт червоний→зелений.
​

ratingcolormain.js — Найпростіший з трьох. Не чіпає логіку рейтингів — лише фарбує вже існуючі бейджі (.card__vote, .full-start__vote тощо) у фіксовані кольори по порогам: ≥8 зелений, ≥7 салатовий, ≥6 гірчичний, і т.д. до червоного. Запускається через setInterval кожну секунду + при події app:ready

2rankds.js Комплексний плагін для Lampa, який розширює відображення рейтингів — підтягує дані з зовнішніх API і показує їх як у деталях фільму, так і на постерах.
Джерела рейтингів:
MDBList API — агрегатор (IMDb, Metacritic, Rotten Tomatoes, Popcorn)
OMDb API — вікові обмеження та нагороди (Оскар, Еммі)
CUB — рейтинг на основі реакцій (злитий з cubreiting.js)
TMDB — через вбудований Lampa-клієнт
Що відображає на сторінці фільму:
Рейтинги IMDb, Metacritic, Rotten Tomatoes, Popcorn, CUB з іконками
Середній агрегований рейтинг (зірка)
Нагороди: Оскар 🏆, Еммі, інші
Вікові обмеження (G / PG-13 / R → 3+ / 13+ / 17+)
Опціонально — бейджі прямо на постерах (ratings_poster_badges: true)
Технічно:
Кеш рейтингів у localStorage на 3 дні
Поліфіли Promise, fetch, localStorage — працює навіть на старих TV-браузерах
Налаштування через Lampa.SettingsApi (розділ «Рейтинги»), застосовуються без перезавантаження
Потребує власних API-ключів MDBList та OMDb

menu_editor.js Плагін для Lampa, який дозволяє редагувати всі три меню додатку прямо з налаштувань — змінювати порядок і приховувати пункти.
Що вміє:
Ліве меню — переміщення пунктів вгору/вниз, приховування окремих розділів
Верхнє меню (шапка) — перестановка іконок (пошук, профіль, сповіщення тощо), додавання кнопок «Перезавантаження» та «Очистити кеш»
Меню налаштувань — зміна порядку розділів у Settings, приховування непотрібних
Панель навігації — окремий тогл для її повного приховування (актуально для телефонів)
Всі зміни зберігаються в Lampa.Storage і відновлюються при кожному запуску через слухачі подій menu:end та activity:start.

buttons_1.js (v1.4.0) — розширена версія редактора кнопок. Все те саме що і buttons.js, але додає папки: можна групувати кнопки в папки, редагувати порядок всередині них, перейменовувати. Також дозволяє задавати індивідуальний режим відображення для кожної кнопки окремо (стандарт / тільки іконка / з текстом). ES5-сумісний з повним набором поліфілів.
​

buttons_2.js (v2.1) — альтернативна версія редактора без папок, але з перейменуванням кнопок через текстове поле вводу. Ключова відмінність — стабільна генерація ID кнопок (generateStableButtonId) на основі класів, тексту, href та data-action, що усуває проблему «зламаного порядку після перезапуску». Також підтримує кнопку «Options (…)» яку інші версії ігнорують.
