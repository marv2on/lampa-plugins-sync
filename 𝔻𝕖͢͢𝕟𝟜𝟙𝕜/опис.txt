netflix_premium_style.js Netflix Premium Style v8 — масштабна тема оформлення для Lampa у стилі Netflix. Включає:

Логотипи фільмів — завантажує з TMDB API з анімованою заміною заголовка, кешуванням і вибором мови
Hero-сторінка — повноекранний backdrop без градієнтів, динамічний туман при скролі, приховано постер
Картки — GPU-анімація масштабування при фокусі, зсув сусідніх карток, кольорові бейджі рейтингу (зелений/жовтий/червоний), кольоровий бейдж якості
Бокове меню — glassmorphism (blur + прозорість), червона ліва лінія активного пункту
Адаптивність — окремі стилі для телефону, планшета, TV та 4K
Налаштування — акцентний колір, шрифт, масштаб карток, ширина/прозорість меню, висота логотипу, відображення рейтингів (з можливістю приховати KP) тощо
Оптимізовано для слабких Smart TV (Tizen/WebOS/Vidaa) — вимикає blur автоматично (https://ourbeerdelivery.github.io/pluginlmp/netflix_premium_style.js)
