upcoming.js Плагин для платформы Lampa добавляет раздел с трейлерами, интегрируясь с API The Movie Database (TMDB). Он: Загружает данные: Использует TMDB API для получения информации о фильмах и сериалах в категориях "Популярные фильмы", "В прокате", "Скоро в кино", "Популярные сериалы", "Новые сериалы и сезоны", "Скоро на ТВ". Фильтрующий контент: Исключает нежелательные жанры и ключевые слова, проверяет даты релиза и рейтинги. Отображает карты: Создает карты с трейлерами через объект Trailer, показывая название, дату премьеры, рейтинг, язык трейлера и для сериалов номер сезона/эпизода. Язык трейлера: предпочтение отдается языку интерфейса (UA, Ru, EN), а при отсутствии дубляжа – EN. Интерактивность: Позволяет просматривать трейлеры на YouTube, листать списки, переходить к подробной информации. Адаптивность: Подстраивается под язык и размер экрана, используя шаблоны Lampa. Начальный код был от CUB (https://mastermagic98.github.io/l_plugins/upcoming.js)

Плагін для Lampa, який дозволяє редагувати кнопки на сторінці фільму/серіалу — змінювати порядок, приховувати непотрібні та вибирати режим відображення.
Що вміє:
Перестановка кнопок (Онлайн, Торренти, Трейлери, Обране, Закладки, Реакції тощо) вгору/вниз
Приховування окремих кнопок
3 режими вигляду: стандартний / тільки іконки / з текстом
Кнопки MODSS та Showy завжди залишаються першими (пріоритет)
Анімація появи кнопок з затримкою (fade-in з кроком 0.08s)
Замінює стандартну іконку Lampac на власну
Налаштування зберігаються в Lampa.Storage (button_custom_order, button_hidden). Редактор відкривається через спеціальну кнопку «✏️» прямо на сторінці фільму. Зовнішніх залежностей немає. (https://mastermagic98.github.io/interface/+buttons.js)
