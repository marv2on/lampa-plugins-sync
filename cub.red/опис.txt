youtube-player.js — змінює зовнішній вигляд плеєра Lampa на стиль YouTube/Netflix. Перекомпоновує панель керування: групує кнопки у заокруглені блоки, виносить назву серіалу/фільму окремим великим заголовком, прибирає зайві елементи, додає напівпрозорий градієнт-оверлей при відображенні панелі. Не працює на мобільних пристроях. (https://cub.red/plugin/youtube-player)

iptv.js — оригінальна версія того ж плагіну. Домен посилань захардкоджений як cub.rip, повні переклади UI на 6 мов (ru/uk/be/zh/pt/bg), наявні методи-заглушки emit()/use(). (https://cub.red/plugin/iptv)
Що робить плагін:
Завантажує M3U-плейлисти (через CUB API або напряму)
Відображає список каналів з групами та пошуком
Показує EPG (телепрограму) через CUB API або кастомний XMLTV
Підтримує catchup (архів) з різними типами: append, flussonic, xc, shift
Зберігає вибране, заблоковані канали та параметри в IndexedDB
Інтегрується з Lampa Player (HUD, перемикання каналів, таймлайн програми)
